# Minecraft Plugin Builder

Visual drag-and-drop Minecraft plugin builder. Create Bukkit/Paper plugins without writing code.

## Quick Start

### Backend
```bash
cd backend
pip install -r requirements.txt
python run.py
```

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Requirements
- Python 3.10+
- Node.js 18+
- Maven (for building plugins)
- Java 21 (for compiled plugins)
